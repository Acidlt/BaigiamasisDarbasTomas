Movie Recommendation System

It's a simple system where a user can give ratings to their favorite movies and get recommendations based on those ratings or just simply based on a movie.

The system has 4 recommendation algorithms.

Content-Based filtering using Cosine Similarity

This filtering is based on Cosine-Similarity. What that means is that this algorithm calculates how similar movies are based on their features.

Collaborative filtering using K-NN

This filtering is based on k-Nearest Neighbors algorithm. What that means is that this algorithm finds nearest neighbors in this cases similar users. Then it predicts ratings for specific movies based on those neighbors.

Collaborative filtering using neural networks.

This filtering is based on using neural networks. What that means is that a neural network model is used for finding latent factors for both users and movies. Then it can predict ratings based on user and movie latent factors.

Hybrid filtering

This filtering is based on a hybrid approach. What that means is that it gives weights to K-NN and Neural network methods and combines the recommendations from those two methods.
